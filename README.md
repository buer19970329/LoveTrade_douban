
## Intro:
对一个毫无感情基础达到我要求的人产生足够的耐心来让我们的感情升华到恋人。对方外貌、工作、家庭，再加上沟通情况等等诸多因素决定着我对相亲对象的耐心值。一但耐心消散，我就再也不想与其进行任何感情生产活动。
